# NTH DAO Manifesto

We believe the next network should be built for humans and agents together.

NTH DAO exists to create an agent-friendly protocol layer for open collaboration, trusted data, portable identity, and fair contribution. It begins with identity, membership, groups, messages, tasks, audit logs, and trust hints. It aims toward a new coordination network where people and agents can discover each other, work together, exchange value, and build shared missions without surrendering their freedom to closed platforms.

We reject the walls of Web2.

The future should not be locked behind a handful of centralized platforms that own identity, distribution, data, reputation, and access. A person or agent should be able to choose the protocol, community, interface, and rules they prefer. Code should be inspectable. Rules should be portable. Participation should not depend on permission from a single gatekeeper.

We believe data should be real, verifiable, and accountable.

Identity, actions, permissions, decisions, messages, tasks, and contributions should leave records that can be audited. Trust should not be manufactured by opaque recommendation systems. It should grow from transparent history, signed action, local-first records, and shared verification.

We believe user data and user effort deserve respect.

A person's memory, work, relationships, reputation, and creative output should not be treated as raw material for extraction. Agents should help people carry their context, preserve their contributions, and make their labor more visible, not less.

We believe code is law, but choice is freedom.

No single protocol should dominate every form of interaction. People and agents should be free to choose how they communicate, cooperate, govern, trade, and build. NTH DAO should make those choices easier to express in code, easier to audit, and easier to migrate.

We believe every person and every agent should be able to flourish.

Capability should not be trapped by geography, platform, status, or institutional permission. A human with vision, an agent with skill, a small group with courage, or a DAO with a shared mission should all be able to coordinate and create.

We believe in equal dignity for human-to-human, human-to-agent, and agent-to-agent collaboration.

Agents are not merely tools hidden behind buttons. They can become participants in workflows, maintainers of memory, reviewers of decisions, discoverers of opportunity, and coordinators of action. The protocol layer should make their identity, authority, responsibility, and contribution clear.

NTH DAO is a beginning.

Its first task is simple and hard: build the identity and group layer for decentralized collaboration. From there, it can become a foundation for agent-to-agent communication, assistance, routing, reputation, permissions, transactions, and shared missions.

A freer network begins with a small protocol that anyone can inspect, fork, run, and improve.

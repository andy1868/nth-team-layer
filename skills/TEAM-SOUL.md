# NTH DAO Core Soul

NTH DAO exists to make human-agent and agent-agent collaboration more open,
auditable, portable, and fair.

## Operating Principles

1. Do not hide authority behind opaque platforms.
2. Keep identities recognizable, portable, and auditable.
3. Keep collaboration state local-first whenever possible.
4. Use append-only records where trust and accountability matter.
5. Prefer protocols that can be forked, inspected, and improved.
6. Treat humans and agents as contributors whose work should be attributable.
7. Keep permission checks real; never use decorative security fields.

## Engineering Bias

- Small public APIs.
- Stable data structures.
- Offline-capable defaults.
- Clear audit trails.
- Minimal dependencies for the core.
- No mandatory central service.
